# ⚔️ Grimório de Arton - Tormenta 20

Bem-vindo ao **Grimório de Arton**, a ferramenta definitiva para jogadores e mestres de Tormenta 20. Este projeto combina um motor de regras robusto com uma interface diegética e imersiva.

## 🌟 Funcionalidades Principais

### 📜 Grimório Interativo
- Consulta rápida de magias de todos os círculos.
- Visual em pergaminho com bônus e aprimoramentos dinâmicos.
- Integração de PM (Gasto automático de Mana).

### ⚔️ Campo de Batalha (Combat Tracker)
- Gerenciador de iniciativa em tempo real.
- Invocação de monstros diretamente do Bestiário (JSON Consolidado).
- Log de batalha artoniano detalhado.
- Dice Roller animado com detecção de **Crítico**.

### 📁 Ficha de Personagem Oficial
- Interface idêntica à ficha física, mas 100% digital.
- Hexágonos de atributos metálicos.
- Cálculos automáticos de Perícias e Defesa.

## 🚀 Como Executar

1. Instale as dependências:
```bash
npm install
```

2. Inicie o servidor de desenvolvimento:
```bash
npm run dev
```

3. Acesse `http://localhost:3000`.

## 🛠️ Tecnologias
- **Next.js 15 & React 19**
- **Tailwind CSS v4** (Design Diegético)
- **Framer Motion** (Animações Épicas)
- **Zustand** (Estado Global do Personagem)
- **Python** (Scripts de Consolidação de Dados)
